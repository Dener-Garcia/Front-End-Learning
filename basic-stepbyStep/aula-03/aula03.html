Aula 03 - Condicionais, se acontecer algo fazemos algo

If se uma condição for verdadeira faça isso... seu código

Vamos criar um código para verificar se você é maior de idade

  • let idade = 0;
  • idade = 19;
  • if (idade > 18) {console.log(idade, "você é maior de idade")} // se idade eh maior que 18
  • else {console.log(idade, "você é de menor"} // se nao for executa essa linha

Tipos de Condicionais

  • == igual *cuidado para não usar o = sozinho, senão você está atribuindo um valor a variável
  • < menor
  • > maior
  • <> diferente
  • != ou <= maior ou igual e menor ou igual

    • Diferenças entre o == e o ===

    Vamos supor que você quer comparar a idade de alguem é igual a 20

    • let idade = 30;
    • idade = "20"; // Observe que colocamos um string no lugar do numeral 20
    • if (idade2 == 20){console.log(idade2,"sua idade é 20 anos")}; // repare que ele mostrou o código mesmo tendo uma string

    Para resolver isso temos o === onde ele é usado para ser mais restritivo. Repita o código acima mas usando o === e verifique se vai entrar na condição do seu IF

    Multi condicionais varios If na mesma logica

    vamos somar as idades 1 e 2 com condicões dentro do mesmo if

    • idade2 = 15; // atribui 15 a idade2
    • idade3 = 58; // atribui 58 a idade3
    • resultado = idade2 + idade3; // resultado armazena a soma de idade2 e 3
    • if(resultado >= 18) {if (resultado < 60){ // repare que um if está dentro do outro
    • console.log(resultado, "somando as idades você é um adulto") }} // repare que cada chave fecha um if

    Multi condicionais com AND e OR

    Agora vamos usar o if e gerar condicionais de and e or dentro de um único IF

    • idade2 = 10; // atribui 10 a idade2
    • idade3 = 20; // atribui 20 a idade3
    • resultado = idade2 + idade3; // resultado recebe a soma das idades
    • if ((resultado > 18 && resultado < 60) || resultado == 18){ // repare na condição de And e logo após condição de OR, voce pode priorizar as condições colocando um outro (parenteses)
    • console.log(resultado, "Resultado indica mais de 18 anos")}

    Condicional dupla if else

    Agora vamos usar o if e gerar um else if para filtrar o código dentro de um único IF

    • idade2 = 60; // atribui 10 a idade2
    • idade3 = 8; // atribui 8 a idade3
    • resultado = idade2 + idade3; // resultado recebe a soma das idades
    • if (resultado < 18){ // Abrimos o IF mestre e a chave
    • console.log("Você é de menor") // se o primeiro if ja for atendido ja exibe esse resultado
    • } else if (resultado >= 18 && resultado < 60){ // fechamos a chave do primeir if e criamos um else if (ou se) minha verificação
    • console.log("Você é de menor") // se o esse else if ja for atendido exibe esse resultado
    • else if (resultado > 60){ // fechamos a chave do primeir if e criamos um else if (ou se) minha verificação
    • console.log("Você está idoso") // se o último else if for atendido exibe esse resultado
