Concatenar strings (juntar palavras)

  • let nome = "1Nome";
  • let sobreNome = "2sobrenome";
  • console.log(nome + ' ' + sobreNome); // mostra o nome um espaço e o sobre nome
  • console.log(nome, sobreNome); // mostra o nome seprado do sobre nome sem precisar do espaço devido a virgula

Template string

Muito parecida com a string normal porém tem algumas vantagens como por exemplo fazer pequenos códigos javascript dentro dela mesmo

  • let idade = 20;
  • let idadeCompleta = `Idade: ${idade} anos`; // template strings devem ser usadas aspas simples `` dentro dela pode fazer operações lembre de usar as chaves {} tambem
  • console.log(idade); // mostra a idade com os espaços e o valor da variável idade
  • console.log(nome, sobreNome); // mostra o nome seprado do sobre nome sem precisar do espaço devido a virgula
