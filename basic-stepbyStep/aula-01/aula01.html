Aula 01 - Variaveis, tipo de variaveis e console log

O let e o var são utilizados para criar variáveis

let nome = "seu nome aqui" - Isso é uma variável do tipo string repare nas aspas do nome

let idade = 24 - Isso é uma variável tipo inteiro

  • console.log("Escreva aqui algo para mostrar no console");
  • console.log(nome, idade) // mostra as variváveis nome e idade no console

Exercício 1

A- Crie uma variável de nome carro e mostre ela no console com o nome de Ferrari

B- Crie uma variável para armazenar o preço de um bolo, que custa R$19,90, exiba no console
